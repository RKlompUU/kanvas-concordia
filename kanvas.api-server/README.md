## Description

Back end for the white label store for selling NFTs.

Built with [Nest](https://github.com/nestjs/nest) framework.

Please note that yuou must use Yarn, not NPM to install dependencies.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
